
Run api NestJS server:
npm run start:dev

Run frontend Angular:
ng serve

Add static folder
https://docs.nestjs.com/recipes/serve-static
npm install --save @nestjs/serve-static

Install mongoose:
npm install --save @nestjs/mongoose mongoose
